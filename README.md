# RISC-V Tower of Hanoi

Simple C++ program to solve the Tower of Hanoi problem with ASCII output.

## Build
```bash
g++ -std=c++17 hanoi.cpp -o hanoi
```

## Run
```bash
./hanoi
```

## Output
- Shows initial state  
- Displays key moves  
- Prints total moves  

## Files
- hanoi.cpp  
- output.txt
